# 3.1 Security Control Types

Security controls are categorized by **how** they are implemented and **what** they do.

## Control Categories (The "How")
* **Managerial (Administrative):** Focused on design and strategy (e.g., risk assessments, security policies).
* **Operational:** Implemented by people and processes (e.g., security awareness training, backup procedures).
* **Technical:** Implemented in systems via hardware or software (e.g., firewalls, encryption, EDR).
* **Physical:** Controls that limit physical access (e.g., fences, locks, badges).

## Control Types (The "What")
* **Preventive:** Stops a security event before it happens (e.g., Firewall, IPS).
* **Detective:** Identifies an event as it happens or after the fact (e.g., IDS, Log monitoring).
* **Corrective:** Reverses the impact of an event (e.g., Backups, patching a system after a breach).
* **Deterrent:** Discourages an attacker (e.g., Warning signs, security cameras).
* **Compensating:** An alternative control used when the primary control isn't feasible (e.g., Using a hardware token because biometric scanners are too expensive).
