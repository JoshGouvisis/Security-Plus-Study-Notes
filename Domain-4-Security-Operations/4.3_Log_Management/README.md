# 4.3 Log Management
* **Aggregation:** Centralizing logs from diverse sources (Firewalls, Servers, Endpoints) into a SIEM for analysis.
* **Log Types:**
    * **System Logs:** Record OS events and startup/shutdown activities.
    * **Security Logs:** Track logons, logoffs, and access to secure files.
* **NTP (Network Time Protocol):** Essential for log correlation; all logs must have synchronized timestamps to reconstruct an incident timeline.
